# Diseño de Factura – Monday Shop

## Descripción
Este proyecto corresponde al **diseño visual de una factura comercial**, desarrollado como actividad académica.  
El trabajo presenta una **maqueta estática** que muestra la estructura y presentación de una factura, sin funcionalidades dinámicas ni cálculo automático.
El objetivo del proyecto es representar de forma clara y ordenada la información de una factura, incluyendo datos del cliente, datos de la empresa, listado de productos y totales.

## Contenido del proyecto
El diseño de la factura incluye:
- Nombre y logotipo de la empresa
- Datos del cliente
- Número de factura y fecha
- Tabla de productos con:
  - Imagen referencial
  - Cantidad
  - Descripción
  - Precio unitario
  - Total
- Subtotal, IVA y total a pagar
- Observaciones finales
